SW1 es el switch de distribución principal en la jerarquía de capa 2 por eso además de crear las VLANs, se ha configurado como el Puente Raíz (Root Primary) de Spanning Tree para evitar bucles de red este también inicia la agregación de enlaces mediante LACP en modo activo, negociando un canal de fibra o cobre virtual (EtherChannel) con el SW2 para aumentar el ancho de banda y la redundancia.

- BLOQUE SSH 

enable
configure terminal

ip domain-name campus.local
crypto key generate rsa general-keys modulus 1024

ip ssh version 2

username 20241375 privilege 1 secret Fe2024
username opelegrino privilege 15 secret Fe2024

enable secret Fe2024

line console 0
password Fe2024
login
exit

line vty 0 4
transport input ssh
login local
exit

Este bloque establece la seguridad administrativa del equipo por eso en lugar de usar Telnet que envía datos en texto plano, se activa SSH versión 2 con cifrado RSA para proteger las sesiones de configuración remota. Además, define usuarios locales con diferentes niveles de privilegios y asegura el acceso físico a la consola y el acceso privilegiado mediante las contraseñas cifradas.
